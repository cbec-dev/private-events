# Eventy

Aplicación web simple de eventos, cumple con las funcionalidades más básicas de Eventbrite, cuenta con registro y sesiones de usuarios, aùn es necesario corregir y agregar muchas validaciones.

Construida con:

* Rails 5.2 
* Devise
* Bootstrap
* Flatpickr
* Rails-i18n
* Simple form
